# PrepMaster - 800+ Curated QA Interview Questions

## Overview
PrepMaster is a comprehensive, free, and open-source interview preparation platform with 800+ curated QA interview questions. Master Selenium, Java, API Testing, TestNG, Cucumber, and more!

## Features
- **800+ Curated Questions** - Questions collected from various organizations' interviews
- **Multiple Topics** - Selenium, Java, TestNG, Cucumber, API Testing, Maven, Jenkins, Docker, Git, and more
- **Difficulty Levels** - Basic, Intermediate, and Advanced questions
- **Progress Tracking** - Mark questions as done, revise, or star for bookmarking
- **Fast Search** - Search across all topics and questions
- **Responsive Design** - Works on desktop and mobile devices
- **Free & Open Source** - 100% free, no ads, no login required

## Topics Covered
- **Automation Testing** - Selenium, WebdriverIO, Playwright, Appium
- **Testing Frameworks** - TestNG, Cucumber, BDD
- **Java & OOP** - Core Java, OOPS concepts, Collections, Strings, Exceptions
- **API Testing** - REST API, SOAP, GraphQL, API Testing tools
- **DevOps** - Git, Maven, Jenkins, Docker, CI/CD
- **Databases** - SQL, JDBC, Apache POI, Data Driven Testing
- **And More** - Soft skills, Agile, Performance Testing, Security

## How to Use
1. Visit the website to start browsing questions
2. Use the search bar to find questions by topic or keyword
3. Filter by difficulty level (Basic, Intermediate, Advanced)
4. Mark questions as Done, Revise, or Star (bookmark)
5. Track your progress with the progress dashboard

## Live Deployment
This repository contains all the files needed for deploying PrepMaster for free.

### Deploy on GitHub Pages
1. Fork this repository
2. Go to Settings > Pages
3. Select "main" branch as source
4. Your site will be available at `https://yourusername.github.io/prep-master`

### Deploy on Other Platforms
- **Netlify** - Connect your GitHub repo, deploy for free
- **Vercel** - Connect your GitHub repo, deploy for free
- **Firebase Hosting** - Upload the files, deploy for free
